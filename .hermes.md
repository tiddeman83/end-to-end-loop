# Hermes Project Context

This repository is ready for Hermes Agent maintenance. It contains an Agent Skills
compatible `SKILL.md`, validation script, CI workflow, research paper draft, and
DevBoss handoff instructions.

## Hermes Operating Rules

- Load `SKILL.md` before making coding or release changes.
- Enable `skills.write_approval: true` before allowing Hermes to update installed
  skill copies.
- Use `AGENTS.md` as the general project bridge and `handoff/` for DevBoss office
  setup.
- Treat the user as Supervisory Board chair. New release plans require board
  approval before implementation.
- Route board decisions through the user's AI Assistant into Todoist, then mirror
  them back into the DevBoss decision log.
- Do not perform Firebase deploys until the user provides the environment and
  explicit deploy approval.

python3 scripts/validate_skill.py .
